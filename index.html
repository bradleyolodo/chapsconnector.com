CHAPPS CONNECTOR

Open protocol for connecting Wallets.

WHAT IS CHAPS CONNECTOR
(QR Codes & Deep Links)
chaps connector is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking.

HOW IT WORKS
chaps Connectives connects web applications to supported mobile wallets. chapsConnector session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).

PROTOCOL
chaps Connect is not an app, but an open protocol to communicate securely between Wallets and Dapps (Web3 Apps).
The connection is initiated by one peer displaying a QR Code or deep link with a standard WalletConnect URI and is established when the counter-party approves this connection request.

SECURE ENCRYPTION
The protocol establishes a remote connection between two apps and/or devices. These payloads are symmetrically encrypted through a shared key between the two peers.

Wallets

Multiple iOS and Android wallets support the Connector protocol. Interaction between mobile apps and mobile browsers are supported via mobile deep linking.
